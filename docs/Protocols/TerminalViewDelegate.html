  • sizeChanged(source:newCols:newRows:)

    The client code sending commands to the terminal has requested a new size for the terminal Applications that support this should call the TerminalView.getOptimalFrameSize to get the ideal frame size.

    This is needed for the rare cases where the remote client request 80 or 132 column displays, it is a rare feature and you most likely can ignore this request.

  • requestOpenLink(source:link:params:) Default implementation

    Invoked in response to the user clicking on a link, which is most likely a url, but is not mandatory, so custom implementations receive a string, and they can act on this as a way of communciating with the host if desired. The default implementation calls NSWorkspace.shared.open() on the URL.
